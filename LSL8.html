The matrix exponential

Recall from last week that if $A$ is diagonalisable (i.e. $A=HDH^{-1}$) then

\[ e^{At} = H\begin{bmatrix}e^{\lambda_1t}&&&\\&e^{\lambda_2t}&&\\&&\ddots&\\&&&e^{\lambda_nt}\end{bmatrix}H^{-1} \]

Where $H$ is an invertible matrix and $D$ is the diagonal matrix given by

\[ D = \begin{bmatrix}\lambda_1&&&\\&\lambda_2&&\\&&\ddots&\\&&&\lambda_n\end{bmatrix} \]

whose diagonal entries are the eigenvalues of $A$

In the case were $A$ is not diagonalisable we can use the eigendecomposition to write

\[ \color{red}{\boxed{\color{white}{ e^{At} = \sum_{r=0}^\infty \frac{A^rt^r}{r!} = H\begin{bmatrix}e^{J_{n_1}(\lambda_1)t}&&&\\&e^{J_{n_2}(\lambda_2)t}&&\\&&\ddots&\\&&&e^{J_{n_k}(\lambda_k)t}\end{bmatrix}H^{-1} }}} \]

This follows since

\[ e^{Jt} = \sum_{r=0}^\infty \frac{J^rt^r}{r!} = H\begin{bmatrix}\sum_{r=0}^\infty J_{n_1}(\lambda_1)^r\frac{t^r}{r!}&&&\\&\sum_{r=0}^\infty J_{n_2}(\lambda_2)^r\frac{t^r}{r!}&&\\&&\ddots&\\&&&\sum_{r=0}^\infty J_{n_k}(\lambda_k)^r\frac{t^r}{r!}\end{bmatrix}H^{-1} \]

And

\[ e^{J_{n_i}(\lambda_i)t} = \sum_{r=0}^\infty J_{n_i}(\lambda_i)^r\frac{t^r}{r!}, \quad i=1,\ldots, n_k \]

It follows that if we can compute the matrix exponential of a Jordan block then we can compute the matrix exponential of any matrix

To do this we need to use the fact that for commutative matrices $A, B$ (i.e. $AB=BA$) we have

\[ \color{red}{\boxed{\color{white}{ e^{At}e^{Bt} = e^{(A+B)t} }}} \]

Let us start by writing

\[ J_{n_i}(\lambda_i) = \lambda_i I_{n_i} +K_{n_i} \]

Where

\[ K_{n_i} = \begin{bmatrix}0&1&0&\cdots&\cdots&0\\0&0&1&\ddots&&\vdots\\ 0&0&0&1&\ddots&\vdots\\ \vdots&\ddots&0&0&1&0\\\vdots&&\ddots&0&0&1\\\ 0&\cdots&\cdots&0&0&0 \end{bmatrix}\in\mathbb{R}^{n_i\times n_i} \]

Then using the result on the previous slide

\[ \begin{align*} e^{J_{n_i}(\lambda_i)t} &= e^{\left(\lambda_i I_{n_i}+K_{n_i}\right)t}\\ &=e^{\lambda_i t}e^{K_{n_i} t} \end{align*} \]

Noting that

\[ \begin{align*} e^{K_{n_i} t}&=\sum_{r=0}^\infty \frac{t^r}{r!}k_{n_i}^r\\ &=\sum_{r=0}^{n-1} \frac{t^r}{r!}k_{n_i}^r \end{align*} \]

This follows since $K_{n_i}$ is nilpotent

Recall that $K_{n_i}^{r}=0$ for $r \geq n_i$

Thus

\[ \begin{align*} e^{K_{n_i}t}&= I_{n_i}+K_{n_i}t + K_{n_i}^2\frac{t^2}{2!}+\cdots +K_{n_i}^{n_i-1}\frac{t^{n_i-1}}{(n_i-1)!}\\ &=\begin{bmatrix}1&0&0&\cdots&\cdots&0\\0&1&0&\ddots&&\vdots\\ 0&0&1&0&\ddots&\vdots\\ \vdots&\ddots&0&1&0&0\\\vdots&&\ddots&0&1&0\\\ 0&\cdots&\cdots&0&0&1 \end{bmatrix}+ \begin{bmatrix}0&t&0&\cdots&\cdots&0\\0&0&t&\ddots&&\vdots\\ 0&0&0&t&\ddots&\vdots\\ \vdots&\ddots&0&0&t&0\\\vdots&&\ddots&0&0&t\\\ 0&\cdots&\cdots&0&0&0 \end{bmatrix}+\begin{bmatrix}0&0&\frac{t^2}{2!}&0&\cdots&0\\0&0&0&\ddots&\ddots&\vdots\\ 0&0&0&0&\ddots&0\\ \vdots&\ddots&0&0&0&\frac{t^2}{2!}\\\vdots&&\ddots&0&0&0\\\ 0&\cdots&\cdots&0&0&0 \end{bmatrix}+\cdots\\ &+~\begin{bmatrix}0&0&0&\cdots&\cdots&\frac{t^{n_i-1}}{(n_i-1)!}\\0&0&0&\ddots&&\vdots\\ 0&0&0&0&\ddots&\vdots\\ \vdots&\ddots&0&0&0&0\\\vdots&&\ddots&0&0&0\\\ 0&\cdots&\cdots&0&0&0 \end{bmatrix} = \color{#00FF00}{\boxed{\color{white}{\begin{bmatrix}1&t&\frac{t^2}{2!}&\cdots&\cdots&\frac{t^{n_i-1}}{(n_i-1)!}\\0&1&t&\ddots&&\vdots\\ 0&0&1&t&\ddots&\vdots\\ \vdots&\ddots&0&1&t&\frac{t^2}{2!}\\\vdots&&\ddots&0&1&t\\\ 0&\cdots&\cdots&0&0&1 \end{bmatrix}}}} \end{align*} \]

Putting this all together


\[ \color{red}{\boxed{\color{white}{ e^{J_{n_i}(\lambda_i)t} = e^{\lambda_i t}\begin{bmatrix}1&t&\frac{t^2}{2!}&\cdots&\cdots&\frac{t^{n_i-1}}{(n_i-1)!}\\0&1&t&\ddots&&\vdots\\ 0&0&1&t&\ddots&\vdots\\ \vdots&\ddots&0&1&t&\frac{t^2}{2!}\\\vdots&&\ddots&0&1&t\\\ 0&\cdots&\cdots&0&0&1 \end{bmatrix} }}} \]

Example 8.1

If

\[ J = \begin{bmatrix}1&&&&&\\&-1&&&&\\&&-1&&&\\&&&-2&1&0\\&&&0&-2&1\\&&&0&0&-2\end{bmatrix} = \begin{bmatrix}J_1(1)&&&\\&J_1(-1)&&\\&&J_1(-1)&\\&&&J_3(-2)\end{bmatrix} \]

Then

\[ e^{Jt} = \begin{bmatrix}e^{J_1(1)t}&&&\\&e^{J_1(-1)t}&&\\&&e^{J_1(-1)t}&\\&&&e^{J_3(-2)t}\end{bmatrix} \]

Here

\[ e^{J_1(1)t}=e^t\quad \text{and}\quad e^{J_1(-1)t}=e^{-t} \quad\text{(One by one blocks)} \]

And, using the formula derived for the matrix exponential of a Jordan block

\[ e^{J_3(-2)t} = \begin{bmatrix}e^{-2t}&te^{-2t}&\frac{t^2}{2!}e^{-2t}\\ 0&e^{-2t}&te^{-2t}\\ 0&0&e^{-2t} \end{bmatrix} \]

And so

\[ \color{red}{\boxed{\color{white}{ e^{Jt} = \begin{bmatrix}e^t&0&0&&&\\ 0&e^{-t}&0&&&\\ 0&0&e^{-t}&&&\\ &&&e^{-2t}&te^{-2t}&\frac{t^2}{2!}e^{-2t}\\ &&&0&e^{-2t}&te^{-2t}\\ &&&0&0&e^{-2t} \end{bmatrix} }}} \]

Example 8.2

Suppose that $A$ is given by

\[ A = \begin{bmatrix}-2&1&-1\\0&0&-2\\-4&2&0\end{bmatrix} \]

And that $A=HJH^{-1}$, where

\[ H = \begin{bmatrix}1&-1&0\\1&-2&0\\1&0&1\end{bmatrix}\quad H^{-1} = \begin{bmatrix}2&-1&0\\1&-1&0\\-2&1&1\end{bmatrix}\quad J = \begin{bmatrix}-2&0&0\\0&0&1\\0&0&0\end{bmatrix} \]

Compute the matrix exponential of $A$

Solution

To start

\[ e^{Jt} = \begin{bmatrix}e^{-2t}&\\&e^{J_2(0)t}\end{bmatrix} = \begin{bmatrix}e^{-2t}&&\\&1&t\\&0&1\end{bmatrix} \]

So that

\[ \begin{align*} e^{At} &= H\begin{bmatrix}e^{-2t}&\\&e^{J_2(0)t}\end{bmatrix}H^{-1}\\ &= \begin{bmatrix}1&-1&0\\1&-2&0\\1&0&1\end{bmatrix}\begin{bmatrix}e^{-2t}&&\\&1&t\\&0&1\end{bmatrix}\begin{bmatrix}2&-1&0\\1&-1&0\\-2&1&1\end{bmatrix} \\ &=\begin{bmatrix}1&-1&0\\1&-2&0\\1&0&1\end{bmatrix}\begin{bmatrix}2e^{-2t}&-e^{-2t}&0\\1-2t&t-1&t\\-2&1&1\end{bmatrix} = \color{red}{\boxed{\color{white}{ \begin{bmatrix}2e^{-2t}+2t-1&1-t-e^{-2t}&-t\\2e^{-2t}+4t-2&2-2t-e^{-2t}&-2t\\2(e^{-2t}-1)&1-e^{-2t}&1\end{bmatrix} }}} \end{align*} \]

Properties of the matrix exponential

Recall that the matrix exponential is given by

\[ \begin{align*} e^{At}&=I+At+\frac{A^2t^2}{2!}+\frac{A^3t^3}{3!}+\cdots\\ &=\sum_{r=0}^\infty \frac{A^rt^r}{r!} \end{align*} \]

Some important properties of the matrix exponential are:

  1. If $\displaystyle A=HJH^{-1}$ then $\displaystyle e^{A}=e^{HJH^{-1}} = He^{J}H^{-1}$

  2. If $\displaystyle A\in\mathbb{R}^{n\times n}$ then $\displaystyle e^{At}\left|_{t=0}\right. = I_n$

  3. $\displaystyle e^{At}\mathbf{x}_0$ is the unique solution of $\displaystyle \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t}=A\mathbf{x} : \mathbf{x}(0)=\mathbf{x}_0$

The last property is of particular interest to us!

Calculating the matrix exponential using Cayley-Hamilton

Let us consider the matrix from Example 8.2, that is

\[ A = \begin{bmatrix}-2&1&-1\\0&0&-2\\-4&2&0\end{bmatrix} \]

It can be shown that the characteristic polynomial of $A$ is given by

\[ \chi_A(t) = t^2(t+2) = t^3+2t^2 \qquad(\text{Exercise}) \]

From which it follows that

\[ \chi_A(A) = A63+2A^2=0\implies A^3=-2A^2 \]

This is just CH theorem

Now, since $\displaystyle A^3=-2A^2$ we have that

\[ \begin{align*} A^4 &=-2A^3 = 4A^2,\quad A^5=4A^3=-8A^2,\\ A^6 &=-8A^3=16A^2,\quad A^7=16A^3=-32A^2,... \end{align*} \]

Thus

\[ \begin{align*} e^{At}&= I_n+At+\frac{A^2t^2}{2!}+\frac{A^3t^3}{3!}+\frac{A^4t^4}{4!}+\frac{A^5t^5}{5!}+\frac{A^6t^6}{6!}+\cdots\\ &= I_n+At+\frac{A^2t^2}{2!}+\frac{(-2A^2)t^3}{3!}+\frac{(4A^2)t^4}{4!}+\frac{(-8A^2)t^5}{5!}+\frac{(16A^2)t^6}{6!}+\cdots\\ &=I_n+At+A^2\left(\frac{t^2}{2}-\frac{2t^3}{3!}+\frac{4t^4}{4!}-\frac{8t^5}{5!}+\cdots\right)\\ &=\color{red}{\boxed{\color{white}{I_n+At+A^2\left(\frac{1}{4}e^{-2t}-\frac{1}{4}+\frac{t}{2}\right)}}} \end{align*} \]

Thus

\[ e^{At} = I_n+At+A^2\left(\frac{1}{4}e^{-2t}-\frac{1}{4}+\frac{t}{2}\right) \]

And since

\[ A^2=\begin{bmatrix}8&-4&0\\8&-4&0\\8&-4&0\end{bmatrix} \]

We have that

\[ \begin{align*} e^{At}&=\begin{bmatrix}1&&\\&1&\\&&1\end{bmatrix}+\begin{bmatrix}-2&1&-1\\0&0&-2\\-4&2&0\end{bmatrix}t+ \left(\frac{1}{4}e^{-2t}-\frac{1}{4}+\frac{t}{2}\right)\begin{bmatrix}8&-4&0\\8&-4&0\\8&-4&0\end{bmatrix}\\ &=\begin{bmatrix}2e^{-2t}+2t-1&1-t-e^{-2t}&-t\\2e^{-2t}+4t-2&2-2t-e^{-2t}&-2t\\2(e^{-2t}-1)&1-e^{-2t}&1\end{bmatrix} \end{align*} \]

Which agrees with our calculations in Example 8.2
