BIBO stability

Let us now return to controlled systems of the form

\[ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t} = A\mathbf{x}+B\mathbf{u}:\quad \mathbf{x}(0)=\mathbf{x}_0 \]

We say that such a system is stable for bounded inputs if for a given initial condition $\mathbf{x}_0$ and nonnegative constant $L$ for which

\[ ||\mathbf{u}(t)|| \leq L \quad \forall t\geq 0 \]

there exists an $M>0$ such that

\[ ||\mathbf{x}(t)|| < M \quad \forall t\geq 0 \]

Thus if the input to the system is bounded then the system is stable

Recalling that the solution to such a system is given by

\[ \begin{align*} \mathbf{x}(t)&=e^{At}\mathbf{x}_0+\int_0^te^{A\tau}\mathbf{f}(t-\tau)\mathrm{d}\tau\\ &=e^{At}\mathbf{x}_0+\int_0^te^{A\tau}B\mathbf{u}(t-\tau)\mathrm{d}\tau \end{align*} \]

We can make the following observations:

  1. If the corresponding free system \[ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t} = A\mathbf{x} \] is a. stable then the control system is stable for bounded inputs

  2. If the corresponding free system is unstable then so is the control system

Now suppose that in addition we consider the output

\[ \mathbf{y}=C\mathbf{x} \]

then we say the system is bounded input bounded output (BIBO) stable if $\mathbf{x}(t)$ in the defintion of bounded input stability can be replaced by $\mathbf{y}(t)$

Again since

\[ \mathbf{y}(t) =C\mathbf{x}(t) = Ce^{At}\mathbf{x}_0+\int_0^tCe^{A\tau}B\mathbf{u}(t-\tau)\mathrm{d}\tau \]

we see that the control system is BIBO stable if the corresponding free system

\[ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t} = A\mathbf{x} \]

is a. stable

Example 19.1

Discuss the BIBO stability of the system

\[ \]

when the matrix $A$ is similar to the following Jordan normal forms

\[ \begin{bmatrix}-1&0&0&0\\0&-3i&0&0\\0&0&3i&0\\0&0&0&-10\end{bmatrix} \qquad \begin{bmatrix}-1&1&0&0\\0&-1&0&0\\0&0&2i-3&0\\0&0&0&-2i-3\end{bmatrix} \qquad \begin{bmatrix}0&1&0\\0&0&0\\0&0&-2\end{bmatrix} \]

Solution

Firstly

\[ e^{Jt} = \begin{bmatrix}e^{-t}&0&0&0\\0&\cos{3t}&-\sin{3t}&0\\0&\sin{3t}&\cos{3t}&0\\0&0&0&e^{-10t}\end{bmatrix}\quad\implies\quad \text{Not BIBO stable} \]

This follows since $\displaystyle e^{Jt}$ is bounded as $t\to\infty$ due to the trigonometric terms

Next

\[ e^{Jt} = \begin{bmatrix}e^{-t}&te^{-t}&0&0\\0&e^{-t}&0&0\\0&0&e^{-3t}\cos{2t}&-e^{-3t}\sin{2t}\\0&0&e^{-3t}\sin{2t}&e^{-3t}\cos{2t}\end{bmatrix}\to 0\quad \text{as}\quad t\to \infty\quad \implies \quad \text{BIBO stable} \]

Since the free system is a. stable the control system is BIBO stable

Finally

\[ e^{Jt} = \begin{bmatrix}1&t&0\\0&1&0\\0&0&e^{-2t}\end{bmatrix}\to\infty \quad \text{as}\quad t\to\infty \]

so the free system is not a. stable and the control system is not BIBO stable

Example 19.2

Discuss the BIBO stability of a controllable and observable system of the form

\[ \begin{align*} \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t}&=A\mathbf{x}+B\mathbf{u}\\ \mathbf{y}&=C\mathbf{x} \end{align*} \]

when $A$ has characteristic polynomial

\[ \chi_A(t) = t^4+\alpha t^3+(\alpha+1)t^2+\alpha t +1 \]

Solution

The Hurwitz matrix is given by

\[ H = \begin{bmatrix}\alpha&\alpha&0&0\\1&1+\alpha&1&0\\0&\alpha&\alpha&0\\0&1&1+\alpha&1\end{bmatrix} \]

Thus

\[ H_1 =\color{magenta}{\boxed{\color{white}{\alpha>0}}} \qquad \color{white}{H_2=\begin{vmatrix}\alpha&\alpha\\1&1+\alpha\end{vmatrix} = \alpha^2+\alpha-\alpha=} \color{#00FF00}{\boxed{\color{white}{\alpha^2>0}}} \]

And

\[ \begin{align*} H_3=\begin{vmatrix}\alpha&\alpha&0\\1&\alpha+1&1\\0&\alpha&\alpha\end{vmatrix}&=\alpha\begin{vmatrix}\alpha+1&1\\\alpha&\alpha\end{vmatrix}-\alpha\begin{vmatrix}1&1\\0&\alpha\end{vmatrix}\\ &=\alpha(\alpha^2)-\alpha^2 = \alpha^2(\alpha-1)>0 \end{align*} \]

And

\[ H_4 = 1\cdot H_3 = \color{magenta}{\boxed{\color{white}{\alpha^2(\alpha-1)>0}}} \]

Thus $H_3$ and $H_4$ are positive if $\alpha-1>0$ or $\alpha>1$ so that for

\[ \alpha\in(1, \infty) \]

we have BIBO stability of the control system

Stabilising via feedback

Whena system is not BIBO stable we can sometimes stabilise it by applying feedback of the form

\[ \mathbf{u} = K\mathbf{x}+\mathbf{v} \]

Then we obtain

\[ \begin{align*} \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t}&=A\mathbf{x}+B\mathbf{u}\\ &=A\mathbf{x}+B\left(K\mathbf{x}+\mathbf{v}\right)\\ &=\left(A+BK\right)\mathbf{x}+B\mathbf{v} \end{align*} \]

Here we have constructed a closed loop system since the control $\mathbf{u}$ depends upon the state $\mathbf{x}$, as opposed to an open loop system

The stability of the closed loop system is determined by the free system

\[ \color{#00FF00}{\boxed{\color{white}{ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t} = \left(A+BK\right)\mathbf{x} }}} \]

Example 19.3

Determine whether the system

\[ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t} A\mathbf{x}+Bu \]

with

\[ A=\begin{bmatrix}1&0&4\\-1&-1&-2\\1&2&-1\end{bmatrix} \qquad B=\begin{bmatrix}1\\0\\0\end{bmatrix} \]

is BIBO stable

Solution

It is sufficient to determine the eigenvalues of $A$

\[ \begin{align*} \chi_A(t) = \begin{vmatrix}1-t&0&4\\-1&-1-t&-2\\1&2&-1-t\end{vmatrix} &= (1-t)\begin{vmatrix}-1-t&-2\\2&-1-t\end{vmatrix}+4\begin{vmatrix}-1&-1-t\\1&2\end{vmatrix}\\ &=(1-t)((1+t)^2+4)+4(-2+(1+t)) = \color{#00FF00}{\boxed{\color{white}{(1-t)(1+t)^2}}} \end{align*}\implies \color{#00FF00}{\boxed{\color{white}{\text{unstable}}}} \]

Example 19.4

Determine a range of $\alpha$ values for which the system in Example 19.3 can be stabilised using feedback of the form

\[ u=K\mathbf{x}+v \]

where

\[ K = \begin{bmatrix}\alpha&0&-4\end{bmatrix} \]

Solution

\[ \begin{align*} A+BK &= \begin{bmatrix}1&0&4\\-1&-1&-2\\1&2&-1\end{bmatrix} +\begin{bmatrix}1\\0\\0\end{bmatrix}\begin{bmatrix}\alpha&0&-4\end{bmatrix}\\ &=\begin{bmatrix}1&0&4\\-1&-1&-2\\1&2&-1\end{bmatrix} +\begin{bmatrix}\alpha&0&-4\\0&0&0\\0&0&0\end{bmatrix} = \color{magenta}{\boxed{\color{white}{ \begin{bmatrix}1+\alpha&0&0\\-1&-1&-2\\1&2&-1\end{bmatrix} }}} \end{align*} \]

Thus the free system is

\[ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t} = \begin{bmatrix}1+\alpha&0&0\\-1&-1&-2\\1&2&-1\end{bmatrix}\mathbf{x} \]

The characteristic polynomial is given by

\[ \begin{align*} |A+BK-tI_3| &=\begin{vmatrix}1+\alpha-t&0&0\\-1&-1-t&-2\\1&27-1-t\end{vmatrix}\\ &=(1+\alpha-t)\begin{vmatrix}-1-t&-2\\2&-1-t\end{vmatrix} = (1+\alpha-t)(t^2+2t+5)\\\\ &\implies \lambda_1=1+\alpha \quad\text{and}\quad \lambda_{2,3}=-1\pm2i \end{align*} \]

Thus, if $\displaystyle \lambda_1=1+\alpha<0\quad$ i.e. \[ \color{#00FF00}{\boxed{\color{white}{ \alpha<-1 }}} \]

the system is stabilised and so is BIBO stable for this range
