Inhomogeneous ODEs (ctd)

Recall from last week that the system

\[ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t} = A\mathbf{x}+\mathbf{f}(t):\quad \mathbf{x}(0)=\mathbf{x}_0 \]

Has solution

\[ \color{red}{\boxed{\color{white}{ \begin{align*} \mathbf{x}(t) &= e^{At}\mathbf{x}_0 +\int_0^te^{A(t-\tau)}\mathbf{f}(\tau)\mathrm{d}\tau\\ &= e^{At}\mathbf{x}_0 +\int_0^te^{A\tau}\mathbf{f}(t-\tau)\mathrm{d}\tau \end{align*} }}} \]

The equivalence of the two formula follows via a simple change of variable

Consider the first integral

\[ \int_0^te^{A(t-\tau)}\mathbf{f}(\tau)\mathrm{d}\tau \]

Making the change of variable $\displaystyle s=t-\tau$ so that $\displaystyle \mathrm{d}s=-\mathrm{d}\tau$ gives

\[ \begin{align*} \int_0^te^{A(t-\tau)}\mathbf{f}(\tau)\mathrm{d}\tau&=\int_0^te^{As}\mathbf{f}(t-s)(-\mathrm{d}s)\\ &=-\int_t^0e^{As}\mathbf{f}(t-s)\mathrm{d}s\\ &=\int_0^te^{As}\mathbf{f}(t-s)\mathrm{d}s=\int_0^te^{A\tau}\mathbf{f}(t-\tau)\mathrm{d}\tau \end{align*} \]

As required

In the above we have use the fact that $\color{red}{\boxed{\color{white}{\int_a^b=-\int_b^a}}}$

Example 13.1

The matrix

\[ A=\begin{bmatrix}-2&1&1\\0&0&-2\\-4&2&0\end{bmatrix}=HJH^{-1} \]

Where

\[ J=\begin{bmatrix}-2&0&0\\0&0&1\\0&0&0\end{bmatrix}\quad H=\begin{bmatrix}1&-1&0\\1&-2&0\\1&0&1\end{bmatrix}\quad H^{-1}=\begin{bmatrix}2&-1&0\\1&-1&0\\-2&1&1\end{bmatrix} \]

Compute the matrix exponential and use it to determine the solution of

\[ \frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t} = A\mathbf{x}+\mathbf{f}(t):\quad \mathbf{x}(0)=\mathbf{x}_0 \]

Where

\[ \mathbf{f} = \begin{bmatrix}0\\1\\0\end{bmatrix} \quad\text{and}\quad \mathbf{x}_0=\begin{bmatrix}0\\0\\0\end{bmatrix} \]

Solution

Recall that

\[ \begin{align*} e^{At}&=He^{Jt}H^{-1}\\ &= \begin{bmatrix}1&-1&0\\1&-2&0\\1&0&1\end{bmatrix}\begin{bmatrix}e^{-2t}&0&0\\0&1&t\\0&0&1\end{bmatrix} \begin{bmatrix}2&-1&0\\1&-1&0\\-2&1&1\end{bmatrix}\\ &=\begin{bmatrix}1&-1&0\\1&-2&0\\1&0&1\end{bmatrix}\begin{bmatrix}2e^{-2t}&-e^{-2t}&0\\1-2t&t-1&t\\-2&1&1\end{bmatrix}\\ &=\color{red}{\boxed{\color{white}{ \begin{bmatrix}2e^{-2t}+2t-1&1-t-e^{-2t}&-t\\2e^{-2t}+4t-2&2-2t-e^{-2t}&-2t\\2e^{-2t}-2&1-e^{-2t}&1\end{bmatrix}}}} \end{align*} \]

Noting that

\[ e^{At}\mathbf{0}=\mathbf{0} \]

And

\[ \begin{align*} e^{At}\mathbf{f}(t)&=\begin{bmatrix}2e^{-2t}+2t-1&1-t-e^{-2t}&-t\\2e^{-2t}+4t-2&2-2t-e^{-2t}&-2t\\2e^{-2t}-2&1-e^{-2t}&1\end{bmatrix} \begin{bmatrix}0\\1\\0\end{bmatrix}\\ &=\color{red}{\boxed{\color{white}{\begin{bmatrix}1-t-e^{-2t}\\2-2t-e^{-2t}\\1-e^{-2t}\end{bmatrix}}}} \end{align*} \]

Thus

\[ \mathbf{x}(t) = e^{At}\mathbf{x}_0+\int_0^te^{A\tau}\mathbf{f}(t-\tau)\mathrm{d}\tau = \int_0^t\begin{bmatrix}1-\tau-e^{-2\tau}\\2-2\tau-e^{-2\tau}\\1-e^{-2\tau}\end{bmatrix}\mathrm{d}\tau \]

So we just need to compute the integral to solve the ODE

We integrate component-wise so that

\[ \begin{align*} \int_0^t\begin{bmatrix}1-\tau-e^{-2\tau}\\2-2\tau-e^{-2\tau}\\1-e^{-2\tau}\end{bmatrix}\mathrm{d}\tau &= \left.\begin{bmatrix}\tau-\frac{\tau^2}{2}+\frac{1}{2}e^{-2\tau}\\2\tau-\tau^2+\frac{1}{2}e^{-2\tau}\\\tau++\frac{1}{2}e^{-2\tau}\end{bmatrix}\right|_0^t\\ &=\begin{bmatrix}t-\frac{t^2}{2}+\frac{1}{2}\left(e^{-2t}-1\right)\\2t-t^2+\frac{1}{2}\left(e^{-2t}-1\right)\\t+\frac{1}{2}\left(e^{-2t}-1\right)\end{bmatrix} \end{align*} \]

Thus

\[ \color{#00FF00}{\boxed{\color{white}{ \mathbf{x}(t) = \begin{bmatrix}t-\frac{t^2}{2}+\frac{1}{2}\left(e^{-2t}-1\right)\\2t-t^2+\frac{1}{2}\left(e^{-2t}-1\right)\\t+\frac{1}{2}\left(e^{-2t}-1\right)\end{bmatrix} }}} \]

Note

  1. We chose to use the second formula since $\mathbf{f}$ is a constant vector
  2. Integration of a vector or matrix is performed component-wise

Actually we could have solved this problem slightly differently

\[ \begin{align*} \mathbf{x}(t)&=\int_0^te^{A\tau}\mathbf{f}(t-\tau)\mathrm{d}\tau\\ &=\int_0^tHe^{J\tau}H^{-1}\mathbf{f}(t-\tau)\mathrm{d}\tau\\ &=H\left(\int_0^te^{J\tau}\mathrm{d}\tau\right)H^{-1}\mathbf{f} \quad \text{(since $\mathbf{f}$ is constant)} \end{align*} \]

Thus in the case where $\mathbf{f}$ is a constant vector, we can integrate $\displaystyle e^{Jt}$ and then just multiply by $H$ and $H^{-1}$

Let's see how this works in practice

Example 13.2

Solve Example 13.1 using the method over slide

Solution

Recall that

\[ J=\begin{bmatrix}-2&0&0\\0&0&1\\0&0&0\end{bmatrix}\quad H=\begin{bmatrix}1&-1&0\\1&-2&0\\1&0&1\end{bmatrix}\quad H^{-1}=\begin{bmatrix}2&-1&0\\1&-1&0\\-2&1&1\end{bmatrix} \]

So that

\[ e^{Jt} = \begin{bmatrix}e^{-2t}&0&0\\0&1&t\\0&0&1\end{bmatrix} \]

It follows that

\[ \mathbf{x}(t) = H\left(\int_0^t \begin{bmatrix}e^{-2\tau}&0&0\\0&1&\tau\\0&0&1\end{bmatrix}\mathrm{d}\tau \right)H^{-1}\mathbf{f} \]

Now

\[ \int_0^t \begin{bmatrix}e^{-2\tau}&0&0\\0&1&\tau\\0&0&1\end{bmatrix}\mathrm{d}\tau = \left.\begin{bmatrix}-\frac{1}{2}e^{-2\tau}&0&0\\0&\tau&\frac{\tau^2}{2}\\0&0&\tau\end{bmatrix}\right|_0^t \]

So that

\[ \begin{align*} \mathbf{x}(t) &= H\left(\int_0^t \begin{bmatrix}e^{-2\tau}&0&0\\0&1&\tau\\0&0&1\end{bmatrix}\mathrm{d}\tau \right)H^{-1}\mathbf{f}\\ &=\begin{bmatrix}1&-1&0\\1&-2&0\\1&0&1\end{bmatrix}\begin{bmatrix}\frac{1}{2}\left(1-e^{-2t}\right)&0&0\\0&t&\frac{t^2}{2}\\0&0&t\end{bmatrix} \begin{bmatrix}-1\\-1\\1\end{bmatrix}\\ &=\begin{bmatrix}1&-1&0\\1&-2&0\\1&0&1\end{bmatrix}\begin{bmatrix}\frac{1}{2}(e^{-2t}-1)\\\frac{t^2}{2}-t\\t\end{bmatrix} = \color{red}{\boxed{\color{white}{ \begin{bmatrix}t-\frac{t^2}{2}+\frac{1}{2}(e^{-2t}-1)\\2t-t^2+\frac{1}{2}(e^{-2t}-1)\\t+\frac{1}{2}(e^{-2t}-1)\end{bmatrix}}}} \end{align*} \]
