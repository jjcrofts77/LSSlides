Jordan normal forms (Ctd)

Just as for the 2 by 2 case there are several cases to work through

Case 1

\[ \chi_A(t) = (t-\lambda_1)(t-\lambda_2)(t-\lambda_3) \]

In this case we have 3 distinct eigenvalues and so $A$ is diagonalisable

Thus

\[ J = \begin{bmatrix}\lambda_1 & 0 & 0\\ 0 & \lambda_2 & 0\\ 0 & 0 &\lambda_3\end{bmatrix} \]
The AM=1=GM for each eigenvalue

Case 2

\[ \chi_A(t) = (t-\lambda_1)(t-\lambda_2)^2 \]
  1. Start by finding an eigenvector $\mathbf{u}_1$ such that $A\mathbf{u}_1=\lambda_1 \mathbf{u}_1$
  2. Next we need to find eigenvectors corresponding to $\lambda_2$
  3. There are 2 cases

sub-case a

\[ \mathrm{dim}\left(\mathrm{Ker}\left[A-\lambda_2 I_3\right]\right) = 2 \]

In this case the AM=2=GM and so the matrix is diagonalisable, that is

\[ J = \begin{bmatrix}\lambda_1&0&0\\0&\lambda_2&0\\0&0&\lambda_2\end{bmatrix}= \begin{bmatrix}J_1(\lambda_1)&0&0\\0&J_1(\lambda_2)&0\\0&0&J_1(\lambda_2)\end{bmatrix} \]

We can find 2 linearly independent eigenvectors corresponding to $\lambda_2$

sub-case b

\[ \mathrm{dim}\left(\mathrm{Ker}\left[A-\lambda_2 I_3\right]\right) = 1 \]

In this case the GM=1<2=AM and so the matrix is not diagonalisable

Its Jordan form is

\[ J = \begin{bmatrix}\lambda_1&0&0\\0&\lambda_2&1\\0&0&\lambda_2\end{bmatrix} = \begin{bmatrix}J_1(\lambda_1)&0\\0&J_2(\lambda_2)\end{bmatrix} \]

We need to solve

\[ \begin{align*} (A-\lambda_2 I_3)\mathbf{u} &= 0\\ (A-\lambda_2 I_3)\mathbf{v} &= \mathbf{u} \end{align*} \]

to find a generalised eigenvector $\mathbf{v}$ and related eigenvector $\mathbf{u}$

We then set $\displaystyle P=\begin{bmatrix}\mathbf{u}_1&\mathbf{u}&\mathbf{v}\end{bmatrix}\implies P^{-1}AP= J$

The order of the vectors in the matrix $P$ is important here

Note

As noted on the previous slide the order of the vectors

\[ P = \begin{bmatrix}\mathbf{u}_1&\mathbf{u}&\mathbf{v}\end{bmatrix} \]

is important

For the above ordering we get

\[ P^{-1}AP = J = \begin{bmatrix}\lambda_1&0&0\\0&\lambda_2&1\\0&0&\lambda_2\end{bmatrix} = \begin{bmatrix}J_1(\lambda_1)&0\\0&J_2(\lambda_2)\end{bmatrix} \]

However, we coud have ordered the vectors as $\displaystyle P = \begin{bmatrix}\mathbf{u}&\mathbf{v}&\mathbf{u}_1\end{bmatrix}$ in which case

\[ P^{-1}AP = J = \begin{bmatrix}\lambda_2&1&0\\0&\lambda_2&0\\0&0&\lambda_1\end{bmatrix} = \begin{bmatrix}J_2(\lambda_2)&0\\0&J_1(\lambda_1)\end{bmatrix} \]

Any other ordering is incorrect

Case 3

\[ \chi_A(t) = (t-\lambda)^3 \]
  1. Start by finding as many eigenvectors as possible
  2. There are 3 cases

sub-case a

\[ \mathrm{dim}\left(\mathrm{Ker}\left[A-\lambda I_3\right]\right) = 3 \]

In this case the AM=3=GM and so

\[ (A-\lambda I_3)\mathbf{u}=0\quad \forall\mathbf{u}\in\mathbb{R}^{3} \]

Therefore

\[ (A-\lambda I_3)=0\implies A = \begin{bmatrix}\lambda&0&0\\0&\lambda&0\\0&0&\lambda\end{bmatrix} =\begin{bmatrix}J_1(\lambda)&0&0\\0&J_1(\lambda)&0\\0&0&J_1(\lambda)\end{bmatrix}=J \]

sub-case b

\[ \mathrm{dim}\left(\mathrm{Ker}\left[A-\lambda I_3\right]\right) = 2 \]

In this instance we know that $J$ will take the form

\[ J = \begin{bmatrix}\lambda&0&0\\0&\lambda&1\\0&0&\lambda\end{bmatrix} \]

and since the largest Jordan block is of dimension 2 we have that

\[ \displaystyle m_A(t)=(t-\lambda)^2 \]

Thus

\[ \mathrm{Ker}\left[(A-\lambda I_3)^2\right] = \mathbb{R}^3 \]

We can find two eigenvectors and so also need to find a generalised eigenvector

To find the generalised eigenvector $\mathbf{v}$ we simultaneously solve

\[ \begin{align*} (A-\lambda I_3)\mathbf{u}&=\mathbf{0}\\ (A-\lambda I_3)\mathbf{v}&=\mathbf{u} \end{align*} \]

Recall $\mathbf{u}$ is a related eigenvector

To finish we need to find a second eigenvector, which can be any vector $\mathbf{w}\in\mathrm{Ker}\left[A-\lambda I_3\right]$ that is linearly independent of $\mathbf{u}$ (i.e. not a multiple of $\mathbf{u}$)

We then construct $P$ as

\[ \begin{align*} P &= \begin{bmatrix}\mathbf{w}&\mathbf{u}&\mathbf{v}\end{bmatrix}\\\\ &\implies \color{#00FF00}{\boxed{\color{white}{ P^{-1}AP = \begin{bmatrix}\lambda &0&0\\0&\lambda & 1\\0&0&\lambda\end{bmatrix} =\begin{bmatrix}J_1(\lambda)&0\\0&J_2(\lambda)\end{bmatrix}=J}}} \end{align*} \]

Remember that the order of the vectors matters here

sub-case c

\[ \mathrm{dim}\left(\mathrm{Ker}\left[A-\lambda I_3\right]\right) = 1 \]

To start compute

\[ \mathrm{Ker}\left[(A-\lambda I_3)^2\right] \]

And recall by the CH theorem that

\[ \mathrm{Ker}\left[(A-\lambda I_3)^3\right] = \mathbb{R}^3 \]

We need to solve

\[ \begin{align*} (A-\lambda I_3)\mathbf{u}&=\mathbf{0}\\ (A-\lambda I_3)\mathbf{v}&=\mathbf{u}\\ (A-\lambda I_3)\mathbf{w}&=\mathbf{v} \end{align*} \]

So that $\displaystyle P=\begin{bmatrix}\mathbf{u}&\mathbf{v}&\mathbf{w}\end{bmatrix}\implies \color{red}{\boxed{\color{white}{ P^{-1}AP=\begin{bmatrix}\lambda &1&0\\0&\lambda &1\\0&0&\lambda \end{bmatrix}=J_3(\lambda)=J}}}$

Example 5.1

Compute the JNF of the matrix \[ A = \begin{bmatrix}-3&-1&1\\-1&-3&1\\-2&-2&0\end{bmatrix} \]

Solution

\[ \begin{align*} \chi_A(t) &= \left|tI_3-A\right|\\ &=\begin{vmatrix}t+3&1&-1\\1&t+3&-1\\2&2&t\end{vmatrix}\\ &=(t+3)\begin{vmatrix}t+3&-1\\2&t\end{vmatrix}-\begin{vmatrix}1&-1\\2&t\end{vmatrix}-\begin{vmatrix}1&t+3\\2&2\end{vmatrix}\\ &= \color{red}{\boxed{\color{white}{t^3+6t^2+12t+8 = (t+2)^3}}} \end{align*} \]

We see that $\lambda=-2$ with AM=3

This is case 3

We start by computing the eigenspace

\[ \mathrm{Ker}\left[(A+2I_3)\right] \]

That is we solve

\[ (A+2I_3)\mathbf{u}=0 \]

Since

\[ (A+2I_3) = \begin{bmatrix}-1&-1&1\\-1&-1&1\\-2&-2&2\end{bmatrix}\longrightarrow \begin{bmatrix}1&1&-1\\0&0&0\\0&0&0\end{bmatrix} \]

We have one equation with three unknowns

\[ \begin{bmatrix}1&1&-1\\0&0&0\\0&0&0\end{bmatrix}\begin{bmatrix}u^{(1)}\\u^{(2)}\\u^{(3)}\end{bmatrix}\implies u^{(1)}+u^{(2)}-u^{(3)}=0 \]

Setting $u^{(3)}=\alpha$ and $u^{(2)}=\beta$ gives $u^{(1)}=\alpha-\beta$ so that

\[ \mathbf{u} = \begin{bmatrix}\alpha-\beta\\\beta\\\alpha\end{bmatrix} \]

And so the eigenspace is given by

\[ \Biggl \{ \alpha\begin{bmatrix}1\\0\\1\end{bmatrix}+\beta\begin{bmatrix}-1\\1\\0\end{bmatrix} : \alpha,\beta\in\mathbb{R} \Biggr \} \]

The eigenvalue $\lambda=-2$ has GM=2<3=AM

Since we only have two eigenvectors we need to find a generalised eigenvector, that is, solve

\[ (A+2I_3)^2\mathbf{v} = 0 \quad\text{but}\quad (A+2I_3)^2 = 0 \implies (A+2I_3)^2\mathbf{v}=0\quad \forall \mathbf{v}\in\mathbb{R}^3 \]

Recalling that a generalised eigenvector $\mathbf{v}$ must come with a related eigenvector $\mathbf{u}$, i.e.

\[ \begin{align*} (A+2I_3)\mathbf{u}&=\mathbf{0}\\ (A+2I_3)\mathbf{v}&=\mathbf{u} \end{align*} \]

And that $\mathbf{v}$ must not be an eigevector, we choose $\mathbf{v}\in\mathbb{R}^3$ such that it does not lie in our eigenspace

For example

\[ \mathbf{v} = \begin{bmatrix}1\\0\\0\end{bmatrix}\qquad \text{(clearly } \begin{bmatrix}1\\0\\0\end{bmatrix}\neq \alpha\begin{bmatrix}1\\0\\1\end{bmatrix}+\beta\begin{bmatrix}-1\\1\\0\end{bmatrix}) \]

The related eigenvector is given by

\[ \mathbf{u} = \begin{bmatrix}-1&-1&1\\-1&-1&1\\-2&-2&2\end{bmatrix}\begin{bmatrix}1\\0\\0\end{bmatrix} = \begin{bmatrix}-1\\-1\\-2\end{bmatrix} \]

Finally, we need another eigenvector which is linearly independent of $\mathbf{u}$ (not a multiple)

The easiest way to do this is to just choose one of the basis vectors from our eigenspace, say

\[ \mathbf{w}=\begin{bmatrix}1&0&1\end{bmatrix}^T \]

So that $P$ becomes

\[ P = \begin{bmatrix}\mathbf{w}&\mathbf{u}&\mathbf{v}\end{bmatrix} = \begin{bmatrix}1&-1&1\\0&-1&0\\1&-2&0\end{bmatrix} \]

It is left as an exercise to show that

\[ \color{red}{\boxed{\color{white}{ P^{-1}AP = \begin{bmatrix}-2&0&0\\0&-2&1\\0&0&-2\end{bmatrix} = \begin{bmatrix}J_1(-2)&0\\0&J_2(-2)\end{bmatrix} = J }}} \]
