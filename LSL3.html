Let us start by recalling some basic facts from linear algebra

  • The characteristic poloynomial $\displaystyle \chi_A(\lambda)$ is given by the $n$th degree polynomial

    \[ \color{red}{\boxed{\color{white}{ \begin{align*} \chi_A(t)&=\left|A-t I_n\right|\\ &=t^n+a_1t^{n-1}+\cdots+a_{n-1}t +a_n \end{align*} }}} \]

  • If

    \[ A\mathbf{u}=\lambda\mathbf{u}\quad \lambda\in\mathbb{R}, A\in\mathbb{R}^{n\times n}\quad \text{and}\quad \mathbf{0}\neq \mathbf{u} \]

    then $\lambda$ is an eigenvalue of $A$ and $\mathbf{u}$ is an eigenvector of $A$

    •

  • The eigenvalues of $A$ are the roots of the characteristic polynomial, i.e. solutions of the equation $\displaystyle \chi_A(\lambda)=0$

    •

  1. The algebraic multiplicity (AM) of an eigenvalue $\lambda_i$ is the number of times $(t-\lambda_i)$ occurrs as a factor of $\displaystyle\chi_A(t)$

    For example

    \[ \chi_A(t) = (t-2)^2(t-1)(t+3)^3 \]

    has 3 distinct eigenvalues $\lambda_1=2$, $\lambda_2=1$ and $\lambda_3=-3$ with respective AMs of 2, 1 and 3

  2. The geometric multiplicity (GM) of an eigenvalue $\lambda_i$ is the dimension of the corresponding eigenspace, i.e. the number of free parameters needed to describe an eigenvector

    3.

  3. $A$ is diagonalisable iff we have that the AM=GM separately for all eigenvalues of $A$, i.e. there exists an invertible matrix $P$ such that

    \[ P^{-1}AP = D = \begin{bmatrix}\lambda_1&&\\&\ddots&\\&&\lambda_n\end{bmatrix} \]

Recall from last weeks lecture that the matrix

\[ A = \begin{bmatrix}1&-1\\1&3\end{bmatrix} \]

is not invertible

This follows since

\[ \chi_A(t) = (t-2)^2 \quad \text{and}\quad \Biggl \{ \alpha\begin{bmatrix}-1\\1\end{bmatrix}: \alpha\in\mathbb{R} \Biggr \} \]

so that $AM=2\neq 1=GM$

It does however have a JNF

  1. For any invertible matrix $Q$, the expression $\displaystyle A=QBQ^{-1}$ is called a similarity transformation and the matrices $A$ and $B$ are said to be similar



Note


  • From 6. we have that a matrix $A$ is similar to a diagonal matrix $D$ iff $AM=GM$ separately for all eigenvalues

  • From the previous lecture we know that there always exists a similarity transformation $P$ such that $\displaystyle A=PJP^{-1}$

  • Here $J$ is the Jordan normal form of the matrix $A$

Example 3.1

The matrices

\[ A_1 = \begin{bmatrix}1&0\\0&1\end{bmatrix}\quad \text{and}\quad A_2 = \begin{bmatrix}1&1\\0&1\end{bmatrix} \]

have the same characteristic polynomial given by

\[ \chi_{A_1}(t) = \chi_{A_2}(t) = (t-1)^2 \]

which implies $\lambda=1$ has AM=2 in both cases

What about the GM?

The eigenspace of the matrix $A_1$ is calculated as

\[ \left(A_1-I_2\right)\mathbf{u}=0\implies \begin{bmatrix}0&0\\0&0\end{bmatrix}\begin{bmatrix}u^{(1)}\\u^{(2)}\end{bmatrix} = \mathbf{0} \]

and so the eigenspace is given by $\mathbb{R}^2$, i.e. $\color{#00FF00}{\boxed{\color{white}{\displaystyle \Biggl \{ \alpha\begin{bmatrix}1\\0\end{bmatrix}+\beta\begin{bmatrix}0\\1\end{bmatrix}: \alpha, \beta \in\mathbb{R} \Biggr \}}}}$

The eigenspace of the matrix $A_2$ is calculated as

\[ \left(A_2-I_2\right)\mathbf{u}=0\implies \begin{bmatrix}0&1\\0&0\end{bmatrix}\begin{bmatrix}u^{(1)}\\u^{(2)}\end{bmatrix} = \mathbf{0} \implies u^{(2)}=0 \]

and so the eigenspace is given by

\[ \Biggl \{ \alpha\begin{bmatrix}1\\0\end{bmatrix}: \alpha\in\mathbb{R} \Biggr \} \]

From the previous slides we see that for the matrix $A_1$ the eigenvalue $\lambda =1$ has $AM=2=GM$ and so $A_1$ is (trivially) diagonalisable

For the matrix $A_2$ the eigenvalue $\lambda =1$ has $AM=2\neq 1=GM$ and so $A_2$ is not diagonalisable

Of course in this simple example these observations could be deduced immediately; however, this will not typically be the case

Cayley-Hamilton Theorem

Theorem 3.1  (Cayley-Hamilton Theorem)

A square matrix $A\in\mathbb{R}^{n\times n}$ satisfies its own characteristic equation, i.e.

\[ \chi_A(A) = 0 \]

The proof of this theorem lies outside the scope of the current course

Example 3.2

Let $\displaystyle A=\begin{bmatrix}1&1\\2&1\end{bmatrix}$ then

\[ \begin{align*} \chi_A(t)&=\begin{vmatrix}1-t &-1\\-2&1-t\end{vmatrix}\\&=(t-1)^2-2 =\color{red}{\boxed{\color{white}{ t^2-2t-1}}} \end{align*} \]
\[ \implies\quad \color{#00FF00}{\boxed{\color{white}{ \begin{align*} \chi_A(A)&=\begin{bmatrix}1&1\\2&1\end{bmatrix}^2-2\begin{bmatrix}1&1\\2&1\end{bmatrix}-\begin{bmatrix}1&0\\0&1\end{bmatrix}\\ &=\begin{bmatrix}3&2\\04&3\end{bmatrix}-\begin{bmatrix}2&2\\4&2\end{bmatrix}-\begin{bmatrix}1&0\\0&1\end{bmatrix}\\&=\begin{bmatrix}0&0\\0&0\end{bmatrix} \end{align*} }}} \]

We can sometimes use the C-H theorem to compute powers of a matrix

Example 3.3

For the matrix $\displaystyle A=\begin{bmatrix}1&2\\3&4\end{bmatrix}$ we have

\[ \chi_A(t) = \begin{vmatrix}1-t&-2\\-3&4-t\end{vmatrix} = t^2-5t-2 \]

So that

\[ A^2-5A-2I_2=0 \implies A^2=5A+2I_2 \]

Now...

Using this result we can work out any power of $A$ in term sof $A$ and $I_2$

Since $\displaystyle\color{#00FF00}{\boxed{\color{white}{A^2=5A+2I_2}}}$ we have that

\[ \begin{align*} A^3&=(5A+2I_2)A = 5A^2+2A = 5(5A+2I_2)+2A = 27A+10I_2\\ A^4&= A^3A=(27A+10I_2)A = 27A^2+10A = 27(5A+2I_2)+10A = 145A+54I_2 \end{align*} \]

etc.

We can even work out the inverse of the matrix

\[ A^2=5A+2I_2 \implies A^{-1}=\frac{A-5I_2}{2} \]

Let's check this

\[ A^{-1}=\frac{1}{2}\begin{bmatrix}-2&2\\3&-1\end{bmatrix}\implies \frac{1}{2}\begin{bmatrix}-2&2\\3&-1\end{bmatrix}\begin{bmatrix}1&2\\3&4\end{bmatrix} = \frac{1}{2}\begin{bmatrix}2&0\\0&2\end{bmatrix} = I_2 \]

as required

Minimal Polynomial

Definition 3.1  (Minimal polynomial)

The minimal polynomial $\displaystyle m_A(t)$ of $A$ is the monic polynomial of least degree such that

\[ m_A(A) = 0 \]


Note

  1. Monic means that the coefficient of the highest power term of $m_A(t)$ is equal to 1
  2. The minimal polynomial of $A$ is unique and divides the characteristic polynomial
  3. This means that the factors of $m_A(t)$ are also factors of $\chi_A(t)$


We can use this last fact to compute $m_A(t)$

Example 3.4

Consider $\displaystyle \chi_{A_2}(t)=(t-1)^2$ where $A_2$ is the matrix from Example 3.1

Then we know $m_{A_2}(t)$ is either

\[ (t-1)^2\qquad \text{or} \qquad (t-1) \]

Let's check

\[ A_2-I_2 = \begin{bmatrix}1&1\\0&1\end{bmatrix}-\begin{bmatrix}1&0\\0&1\end{bmatrix}=\begin{bmatrix}0&1\\0&0\end{bmatrix}\neq 0 \]

It follows that

\[ m_{A_2}(t) = (t-1)^2 \]

In this instance $\color{#00FF00}{\boxed{\color{white}{m_{A_2}(t)=\chi_{A_2}(t)}}}$

Example 3.5

Find the minimal polynomial of the matrix $\displaystyle A_3 = \begin{bmatrix}1&1&0\\0&1&0\\0&0&1\end{bmatrix}$

Solution

Since $A_3$ has eigenvalue $\lambda=1$ with AM=3 we have that

\[ \chi_{A_3}(t) = (t-1)^3 \]

And so $m_{A_3}(t)$ is either

\[ (t-1),\quad (t-1)^2\quad\text{or}\quad (t-1)^3 \]

It should be clear that $A_3-I_3\neq 0$ and that $(A_3-I_3)^3=0$ (why?) and since

\[ (A_3-I_3)^2 = \begin{bmatrix}0&1&0\\0&0&0\\0&0&0\end{bmatrix}^2=0\implies \color{#00FF00}{\boxed{\color{white}{m_{A_3}(t) = (t-1)^2}}} \]
